# Real-Time-Video-Intelligence-System-RTVIS-
RTVIS is a modular, real-time computer vision system that processes video streams to detect, track, and analyze object behavior. The system transforms raw video into actionable intelligence through event detection (intrusion, loitering, line crossing), and exposes APIs and dashboards for monitoring and control.<br><br>


High Level Architecture: Video → Ingestion → Preprocessing → Detection → Tracking → Event Engine → Storage/API/UI
